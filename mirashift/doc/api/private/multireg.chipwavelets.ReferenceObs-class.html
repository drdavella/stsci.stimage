checkOverlap(self, obs)

Determine whether this observation overlaps the current reference mask.

overlapMask(self, obs)

Updates mask of entire observation in output field, IF it is found to overlap observations already in reference mask.

It returns a flag denoting whether the observation overlapped or not and, therefore, whether the mask was updated or not.

It works on entire observations, rather than just chip-by-chip.
